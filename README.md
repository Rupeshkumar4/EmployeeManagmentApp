# 🚀 Employee Management App (Angular)

A modern **Employee Management System** built using **Angular** that helps organizations manage employees, leave balances, and leave requests efficiently.

---

## 📌 Overview

This project provides a clean and intuitive dashboard for HR and employees to:

* Manage employee records
* Track leave balances
* Submit and manage leave requests

---

## ✨ Features

### 👨‍💼 Employee Management

* Add new employees with details:

  * Name
  * Email
  * Contact Number
  * Department
  * Designation
  * Username & Password
  * Role (HR / Employee)
* View employee list in a structured table
* Perform actions:

  * 👁 View
  * ✏️ Edit
  * 🗑 Delete

---

### 📊 Leave Balance Management

* View leave balances in a **card-based UI**
* Supports multiple leave types:

  * Sick Leave
  * Paid Leave
* Displays:

  * Employee name & ID
  * Remaining leaves
  * Last updated date
  * Updated by (Admin/HR)
* Actions available:

  * 👁 View
  * ✏️ Edit
  * 🗑 Delete

---

### 📝 Leave Requests

* Submit leave requests via form:

  * From Date
  * To Date
  * Leave Type
  * Reason
* View leave balance alongside request form
* Tracks:

  * Leave type
  * Count
  * Updated details

---

### 🔐 Role-Based Access

* **HR Role**:

  * Full access to employees
  * Manage leave balances
  * Approve/manage requests
* **Employee Role**:

  * View own leave balance
  * Submit leave requests

---

### 🎨 UI/UX Highlights

* Clean dashboard layout
* Sidebar navigation
* Responsive cards and tables
* Color-coded leave types
* Action icons for better usability

## ⚙️ Tech Stack

* **Frontend:** Angular (v21+)
* **Styling:** CSS / Bootstrap (or custom styles)
* **State Management:** Angular Services
* **Routing:** Angular Router
* **Testing:** Vitest

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone <your-repo-url>
cd EmployeeManagmentApp
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the Application

```bash
ng serve
```

Open in browser:

```
http://localhost:4200/
```

---

## 🏗️ Build

```bash
ng build
```

Production build files will be generated in:

```
dist/
```

---

## 🧪 Testing

### Unit Tests

```bash
ng test
```

### End-to-End Tests

```bash

```
